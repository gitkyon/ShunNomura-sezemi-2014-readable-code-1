eclipceで実行する場合
プロジェクト内に"recipe-data.txt"というファイルを作成し、"オムライス"と記述しUTF-8で保存します。
改行で複数のレシピを入力することが出来ます。

実行するとレシピの内容をコンソールに表示します

仕様6
入力　java Test1 ([レシピid(int)])
出力　レシピidを指定した場合は該当するレシピを表示, レシピidを指定しない場合は全てのレシピを表示
読み込みファイルは"recipe-data.txt"

仕様7 
入力　java Test1 ([レシピid(int)])
出力　レシピidを指定した場合は該当するレシピ（urlつき）を表示, レシピidを指定しない場合は全てのレシピ（urlつき）を表示
読み込みファイルは"recipe-data.txt"

仕様8
入力　java Test1 [ユーザ名(String)] [ファイル名(String)] ([ユーザ名(String)] [ファイル名(String)] …) ([レシピid(int])
出力　レシピidを指定した場合は該当するレシピを表示（ユーザ名は全表示）, レシピidを指定しない場合は全てのレシピを表示（ユーザ名は全表示）
読み込みファイルは"recipe-data.txt"
